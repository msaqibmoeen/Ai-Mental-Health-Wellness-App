Health is a Great Wealth
AI Mental Health Wellness App | Flutter & Firebase

An AI-inspired Mental Health Wellness Mobile Application built using Flutter with Firebase as the backend. This project was developed as part of a Mobile Application Development course and focuses on helping users manage stress, anxiety, low mood, and daily mental fatigue.

⚠️ Note: This app is not a medical replacement. It is a supportive self-care tool designed for everyday mental wellness.

Features
Mental health assessments with smart reports & visual analytics
Guided therapy sessions (breathing, meditation, CBT-based exercises, sleep routines, sound therapy)
Private offline journaling for daily thoughts
Coping cards with calming techniques and reflections
AI-style companion chat (text + voice support)
Emergency support with grounding prompts
Backend & Architecture
Firebase Authentication (secure login/signup)
Cloud Firestore (cloud-based user data)

Motivation

Mental health challenges are increasingly common, especially among students and working professionals. This project was built to:

Encourage daily mental health check-ins
Provide simple, actionable support tools
Ensure user privacy through local-first storage
Make self-care accessible and easy to follow
